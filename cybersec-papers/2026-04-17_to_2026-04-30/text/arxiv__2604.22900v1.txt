Module Lattice Reduction via Optimal Sign Selection

Abstract
We extend the CDPR lattice reduction algorithm from ideal to module lattices, leveraging
the trace orthogonality of the power basis to decompose the module into rank-1 submodules
and applying CDPR independently to each. This base module reduction achieves a Hermite
factor exp( ˜O(√n)) matching the ideal case, with a module reduction factor O(1) independent
of the rank, under a balance hypothesis automatically satisfied for MLWE-distributed bases.
To control precision, we introduce CRT-scaled rounding at totally split primes, reducing
the Gram–Schmidt rounding error and yielding a bounded-precision implementation. We
further reformulate the CDPR sign-selection subproblem as a mixed-integer linear program,
determining the optimal balanced discrepancy to be a universal constant δ∗≈0.4407. All
results build on the class number one condition h+
k = 1 established in Part I of this series.
Keywords: Module lattices, cyclotomic rings, CDPR attack, MILP, lattice reduction, discrep-
ancy theory.

Introduction
Post-quantum cryptography has become a cornerstone of modern cybersecurity, as scalable
quantum computing might threaten various classical public-key cryptosystems. In August 2024,
NIST finalized four post-quantum standards [1, 2, 3]: ML-KEM (FIPS 203), ML-DSA (FIPS
204), SLH-DSA (FIPS 205), and FN-DSA (draft FIPS 206). The first two-and the most widely
deployed-are based on the Module Learning With Errors (MLWE) problem [4, 5, 6] over 2k-th
cyclotomic rings R = Z[ζ2k] ∼= Z[x]/(xn + 1) with n = 2k−1. These standards use module ranks
d ∈{2, 3, 4} and n = 256 (k = 9) [1].
The key secure question is how hard the approximate Shortest Vector Problem (SVP) is on
the module lattices underlying these schemes. For ideal lattices (d = 1), the CDPR [7] exploits
the algebraic structure of cyclotomic rings to solve approximate SVP with factor exp( ˜O(√n)),
an exponential improvement over generic lattice algorithms [8, 9, 10, 11] whose approximation
factors are exp( ˜O(n)). However, extending CDPR to module lattices (d ≥2) has remained open
because the algebraic structure of module lattices is richer.
A key sub-problem in the CDPR pipeline is the sign-selection problem, given the log-embedding
geometry of the ring, assign balanced signs s ∈{±1}|G|−1 to orbit elements so as to minimize the
L∞-discrepancy of the resulting error vector. Prior work used the tower greedy heuristic [7, 12]
and achieved discrepancy Θ(
p
|G| · k). It was widely assumed that this growth was intrinsic to
the lattice geometry.
The journal version of CDPR [12] refined quantum attack on ideal SVP using Stickelberger
relations. Bernstein [13] and Ducas [14] analyzed the precise constants, while Pellet-Mary,
Hanrot, and Stehl´e [15] gave a time-approximation tradeoff. Felderhoff et al. [16] showed that
Ideal-SVP remains hard for small-norm uniform prime ideals. Langlois and Stehl´e [4] proved

The CDPR Pipeline for Ideal Lattices
Before extending CDPR to module lattices, we first review the CDPR quantum attack [7]
on principal ideal lattices (d = 1), establishing the notation and identifying the sign-selection
sub-problem which will be discussed in Section 7.
3.1
Overview of CDPR
Given a principal ideal I = αR ⊆R (an ideal generated by a single element α ∈K×), the CDPR
attack finds a short generator α′ = αϵ (for some unit ϵ ∈R×) with ∥σ(α′)∥∞bounded. The
attack has three phases:
Phase 1: Quantum PIP. Use the quantum algorithm of Biasse–Song [27, 28] to find some one
generator α0 of I in polynomial time.
Phase 2: Log-unit CVP. The set of all generators of I is {α0ϵ : ϵ ∈R×}. In the log-embedding
log |σ(·)| : Rn
>0 →Rn, finding the shortest generator reduces to a Closest Vector
Problem (CVP: given a lattice Λ and a target point t, find v ∈Λ minimizing ∥t−v∥) on
the log-unit lattice Λ = {log |σ(ϵ)| : ϵ ∈R×} ⊂H0, where H0 = {x ∈Rn : P xi = 0}
is the trace-zero hyperplane.
Phase 3: Decode/sign selection. The CVP in Phase 2 is solved by Babai’s round-off algorithm
[40] applied to the cyclotomic unit basis of Λ. The error in this step is controlled by
the sign-selection problem: choosing signs s ∈{±1}|G|−1 to minimize ∥Ms∥∞, where
M is the error matrix derived from the log-sine vector z (Section 3.2).
The overall approximation factor is γ = exp( ˜O(√n)), where the dominant comes from the
distance between the target point and the nearest lattice point in Λ.

Module Reduction via Diagonal CDPR
In this section we extend the CDPR attack from ideal lattices (d = 1) to module lattices (d ≥2)
over 2-power cyclotomic rings. Our algorithm takes as input an R-basis satisfying a balance
hypothesis (Definition 5.1) on its K-Gram-Schmidt vectors, which is automatically satisfied for
bases sampled from the MLWE distribution (Lemma 5.1) and can be enforced on adversarial
inputs by algebraic LLL preprocessing [25, 26].
Given a balanced input basis, the algorithm has three steps. First, we compute the K-linear
Gram-Schmidt decomposition of the input basis. Second, we optionally size-reduce the basis.

CRT-Scaled Rounding
The base algorithm of Section 5 achieves under the C-balance hypothesis on the input basis, with
no dependence on the Phase 1 size-reduction error ρ. Optional coordinate-wise size reduction
(Section 5.3) gives ρ = n/2, which requires O(n)-bit rational arithmetic to manipulate exactly.
In this section we replace coordinate rounding by CRT-scaled rounding: we round in the
finer lattice P −1R at totally split primes, achieving ρ ≤1 and enabling a bounded-precision
implementation in which every ring operation is an n-point NTT over Fps. The approximation
factor is unchanged; the payoff is in complexity and numerical stability.

7.4
Implementation and complexity
A key practical advantage of our approach is that the MILP needs to be solved only once per
cyclotomic ring, not per module instance. The optimal sign vector depends solely on the log-sine
matrix M, which is fixed for a given k, and can be precomputed and stored in O(|G|) bits.
For ML-KEM parameters (k = 9, |G| = 128, Ns = 127), the MILP has only 127 binary
variables, 1 continuous variable, and 257 constraints. Modern solvers such as Gurobi or CPLEX
solve this instance in under seconds on standard desktop hardware. For k = 10 (Ns = 255), the
solve time is under minutes, as in Table 2.
We analyze the asymptotic complexity of the full algoriT: Phase 1: Requires d(d −1)/2
Gram-Schmidt coefficient computations and roundings. Each coefficient computation involves
an inner product in Kd, costing O(dn log n) bit operations via NTT-based ring multiplication.
CRT-scaled rounding adds O(rn log n) operations per coefficient, where r is the number of split
primes. The total classical cost of Phase 1 is O(d2rn log n) bit operations. With precomputed
MILP-optimal signs, the rounding step is reduced to O(n) operations per coefficient, leading to
a total Phase 1 cost of O(d2n log n). Phase 2: Applies the CDPR to d rank-1 submodules. The
dominant cost is the quantum principal ideal problem (PIP) solver of Biasse–Song [27], which
requires poly(n) quantum gates per ideal. The total quantum cost is d · poly(n) gates.

8
Conclusions
We extended the CDPR quantum attack from ideal to module lattices over 2k-th cyclotomic rings.
Our base algorithm (Theorem 5.1) achieves the bound ∥σ(v)∥2 ≤
√
C · γCDPR · det(σ(M))1/(dn)
under a C-balance hypothesis on the input basis, equivalent to Hermite factor exp( ˜O(√n)) and
matching CDPR for d = 1. The balance hypothesis holds for bases sampled from the MLWE
distribution with C = O(1) (Lemma 5.1), and can be enforced on worst-case inputs by algebraic
LLL preprocessing. The key algorithmic idea is to apply CDPR independently to each of the
d rank-1 submodules Mi = M ∩K˜bi and return the shortest of the resulting candidates; the
power-mean inequality converts the per-line bound into a geometric-mean quantity matching
the module determinant.
CRT-scaled rounding (Theorem 6.1) preserves the approximation bound while enabling an
O(d2rn log n)-time bounded-precision implementation via NTT. Finally, MILP-optimal sign
selection (Theorem 7.1 and Corollary 7.1) replaces the tower-greedy CDPR discrepancy Θ(
√
nk)
by a universal constant δ∗≈0.4407, a
√
nk-factor tightening of the implicit constant inside
γCDPR, though not of the exponential exp( ˜O(√n)) itself.
Several open problems remain. First, prove δ∗(k) = O(1) for all k; our computational
evidence covers k ≤10, and a proof would likely exploit hierarchical cancellation in the log-sine
function across the cyclotomic tower. Second, identify δ∗analytically—the value 0.44069 · · ·
does not match obvious known constants, and a closed-form expression would illuminate the
connection to cyclotomic geometry. Third, improve the exponential factor: the exp( ˜O(√n))
factor from Phase 2 is the sole remaining bottleneck, and a better CVP algorithm for the log-unit
lattice would be a major breakthrough.
